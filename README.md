# Scroll_Companion

Go to: edge://extensions/

Click 'Load Unpacked' and then go to instagram.com and you should see it pop up in the bottom right.

### Run it in Chrome

1. Open **`chrome://extensions/`**
2. Turn on **Developer mode** (top right).
3. Click **Load unpacked**.
4. Choose the project folder.
5. Open **`https://www.instagram.com/`** in a new tab.
